Configure Canchek-eID

If you have Administrator access and subscribe to our electronic ID verification product, you can use this screen to customize the email that your customers will see. This email is generated at the time of creating a new eID request at canchek.com.

If you do not have Administrator access, contact customersupport@canchek.com for assistance.

Important!

We generate emails in HTML format and to configure an alternate email, you must be familiar with HTML coding. For assistance, please contact customersupport@canchek.com.

Important!

Any changes to the email generated by Canchek will be live as soon as the changes are saved. Check carefully before creating any new eID requests.

Overriding

This is an example of a customized email format. All standard HTML tags are available. CSS styling is available via the 'style' tag, as shown here.

Tags

You can (and generally should) include the following tags in your template. These will be replaced with the relevant values when the email is generated. The same tag can be used more than once. Tags not listed here, or typed in error, will appear in the email literally.

{androidApp} A link to the Google Play store.
{firmEmail} The email address of the person creating the request.
{firmName} The name of the firm.
{iPadApp} A link to the Apple AppStore.
{pin} The PIN code required to log into our app.
{webApp} A link to canchek.com's eID web product.
